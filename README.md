# Plaksha-Orbital-Pipeline-Simulator
4/5-stage MIPS pipeline simulator with hazard detection, forwarding control, cycle-step simulation, and event logging.


A browser-based MIPS pipeline simulator built for computer architecture courses.
Supports 4-stage (IF ID EX MEM) and 5-stage (IF ID EX MEM WB) pipelines.
Detects RAW and load-use hazards, computes stall cycles with and without
data forwarding, and lets you step through execution cycle by cycle.

No build tools. No dependencies. Open index.html and run.
